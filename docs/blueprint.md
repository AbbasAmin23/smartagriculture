# **App Name**: Kissan Sathi

## Core Features:

- Admin Authentication: Secure authentication for admin users to manage market data.
- Farmer Authentication: Secure authentication for farmers to access real-time data and community features.
- Market Data Management: Admin interface for uploading, updating, and managing vegetable/fruit market data.
- Real-time Price Dashboard: Display up-to-date vegetable/fruit prices in a searchable table for farmers.
- Weather Updates: Show region-specific weather updates using live or mock data and visual representations.
- Price Trend Visualization: Visualize 7-day price trends for each vegetable/fruit using charts.
- Smart Farmer Advice: Provide AI-based advice to farmers based on weather and market trends, with reasoning tool deciding when or if to output a specific advice.
- Community Forum: Enable farmers to post, comment, and perform CRUD operations on posts to foster community interaction.

## Style Guidelines:

- Primary color: Earthy green (#6B8E23) to reflect agriculture and nature.
- Background color: Light beige (#F5F5DC) to create a neutral and calming backdrop.
- Accent color: Warm orange (#FFA500) for call-to-action buttons and highlights, providing contrast and signaling importance.
- Body and headline font: 'PT Sans', a humanist sans-serif, will be used for both headings and body text, for a modern look.
- Code font: 'Source Code Pro' for displaying code snippets.
- Use weather icons from a standard set, such as those from OpenWeatherMap, but re-colored using the colors of the app's color scheme. Add custom icons for vegetable and fruit types and forum functions.
- Use a responsive grid layout to ensure compatibility across devices. Data tables and charts should be clean and easily readable.
- Subtle transitions and animations when updating data or displaying new information.
